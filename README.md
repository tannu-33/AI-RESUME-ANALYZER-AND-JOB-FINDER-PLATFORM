🧠 AI Resume Intelligence & Career Matching Platform

An intelligent web application that analyzes resumes using Natural Language Processing (NLP) techniques and provides data-driven insights along with real-time job and internship opportunities in India.

The system evaluates resumes using multiple scoring metrics such as semantic similarity, ATS compatibility, skill coverage, and resume quality, helping users optimize their resumes while simultaneously discovering relevant career opportunities.

🚀 Features
📄 Resume Intelligence Engine

Upload resumes in PDF or DOCX format

Automatic resume text extraction and processing

Multi-metric resume evaluation including:

Semantic similarity between resume and target role

ATS compatibility scoring

Skill coverage detection

Resume structure and quality analysis

Generates an overall Resume Match Score

🎯 Skill Gap Analysis

Detects skills present in the resume

Identifies missing skills required for the target role

Displays structured lists of:

Matched Skills

Missing Skills

Provides learning resources to help users improve their skill set

✍️ Resume Writing Optimization

Extracts resume bullet points

Detects weak or poorly written bullet statements

Suggests improved, action-oriented bullet points

Performs grammar and writing quality analysis

📊 Resume Performance Dashboard

Displays an interactive score breakdown including:

🧠 Semantic Match Score

🤖 ATS Compatibility Score

🎯 Skill Coverage Score

📄 Resume Quality Score

These metrics are combined to produce a Final Resume Match Score with performance insights.

💼 Live Job & Internship Finder

Fetches real-time job opportunities

Toggle between:

Full-Time Jobs

Internships

Role-based job discovery

Displays job details including:

Job Title

Company Name

Location

Salary (if available)

Direct Apply Link

The job data is fetched using JSearch API via RapidAPI.
